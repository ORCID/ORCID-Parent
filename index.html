ORCID-Mock-API-Web-Application by ORCID

The ORCID Mock API aims to provide an environment where developers can start to integrate ORCID into their systems before a release of the production environment.

Install

Please refer to the mock-api-instructions.pdf for details on how to install and interact with the ORCID Mock API. Please refer to jopmts-instructions.pdf for details on how to install the example ORCID client application.
