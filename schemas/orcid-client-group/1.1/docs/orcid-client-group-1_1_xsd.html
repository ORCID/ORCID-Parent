Element orcid:client-secret
Namespace http://www.orcid.org/ns/orcid
Annotations
 
The client secret to be used by the client for OAuth2 authorization. This
will be generated by ORCID and returned to the administrators of the client system in an XML
file in this format. In other words, this should be omitted in the XML file that is sent to
the ORCID administrators when requesting client creation or client
updates.

This will be generated by ORCID and returned to the administrators of the client system in an XML file in this format.
